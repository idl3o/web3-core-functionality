Introduction to the .NET SDK

The Web3 Crypto Streaming .NET SDK provides a type-safe, intuitive interface for .NET developers to integrate streaming functionality into applications built with C# and other .NET languages. Our SDK supports various .NET platforms including .NET 6+, .NET Framework 4.6.2+, Xamarin, and Unity.

dotnet add package Web3.Streaming.Client

Getting Started

To get started with the Web3 Streaming .NET SDK, you'll need to create a client instance and configure it with your API credentials.

Basic Setup
using Web3.Streaming.Client;
using Web3.Streaming.Client.Config;

// Create client configuration
var config = new Web3StreamingConfig
{
    ApiKey = "your-api-key",
    ApiBaseUrl = "https://api.web3streaming.com/v1",
    Network = NetworkType.Polygon
};

// Initialize the client
var streamingClient = new Web3StreamingClient(config);

// Test connection
bool isConnected = await streamingClient.TestConnectionAsync();
Console.WriteLine($"Connected to Web3 Streaming API: {isConnected}");

User Authentication

The SDK provides methods for authenticating users with Web3 wallets. This is essential for accessing user-specific features such as token balances and stream access rights.

Web3 Wallet Authentication
// Using Nethereum for wallet integration
using Nethereum.Web3;
using Nethereum.Signer;

// Create a challenge message for the user to sign
var challenge = await streamingClient.Auth.CreateChallengeAsync(walletAddress);

// In a real app, the user would sign this with their wallet
// For demo purposes, we'll use a private key (never do this in production!)
var privateKey = "your_private_key_for_testing";
var signer = new EthereumMessageSigner();
var signature = signer.SignMessage(privateKey, challenge.Message);

// Authenticate with the signature
var authResult = await streamingClient.Auth.AuthenticateAsync(new AuthRequest
{
    Address = walletAddress,
    Signature = signature,
    ChallengeId = challenge.ChallengeId
});

// Store the auth token for subsequent requests
streamingClient.SetAuthToken(authResult.Token);

Streaming Content

Once authenticated, users can access streaming content. The SDK provides methods to check access rights, start streams, and retrieve content metadata.

Check Stream Access
// Check if the user has access to a specific content
var contentId = "content_123456";
var accessCheck = await streamingClient.Content.CheckAccessAsync(contentId);

if (accessCheck.HasAccess)
{
    Console.WriteLine($"Access granted until: {accessCheck.ExpiryTime}");
    Console.WriteLine($"Remaining time: {accessCheck.RemainingTime} seconds");
    
    // Get stream URL
    var streamUrl = await streamingClient.Content.GetStreamUrlAsync(contentId);
    Console.WriteLine($"Stream URL: {streamUrl}");
}
else
{
    Console.WriteLine("Access denied. User needs to purchase access.");
}
Start New Stream
// Start a new stream (spends 1 token)
var startStreamResult = await streamingClient.Streaming.StartStreamAsync(new StartStreamRequest
{
    ContentId = contentId
});

if (startStreamResult.Success)
{
    Console.WriteLine($"Stream started. Expires at: {startStreamResult.ExpiryTime}");
    Console.WriteLine($"Transaction hash: {startStreamResult.TransactionHash}");
}
else
{
    Console.WriteLine($"Failed to start stream: {startStreamResult.ErrorMessage}");
}

Wallet Integration

The Web3 Streaming .NET SDK integrates seamlessly with popular Ethereum wallet providers for .NET, such as Nethereum. This enables token transactions and stream purchases.

Connect Wallet with Nethereum
using Nethereum.Web3;
using Nethereum.Contracts;

// Set up Web3 instance with Nethereum
var web3 = new Web3("https://polygon-rpc.com");

// Connect the wallet to our SDK
var walletConnector = new Web3WalletConnector(web3);
streamingClient.SetWalletConnector(walletConnector);

// Now the SDK can interact with the blockchain directly
var balance = await streamingClient.Token.GetBalanceAsync();
Console.WriteLine($"Token balance: {balance}");

Token Management

The SDK provides comprehensive token management capabilities, including balance checking, token transfers, and purchasing streaming credits.

Purchase Tokens
// Purchase tokens with ETH/MATIC
var purchaseResult = await streamingClient.Token.PurchaseTokensAsync(new TokenPurchaseRequest
{
    Amount = 10, // Amount in ETH/MATIC
    GasLimit = 200000
});

if (purchaseResult.Success)
{
    Console.WriteLine($"Purchased tokens successfully!");
    Console.WriteLine($"Transaction hash: {purchaseResult.TransactionHash}");
    Console.WriteLine($"Tokens received: {purchaseResult.TokensReceived}");
}
else
{
    Console.WriteLine($"Token purchase failed: {purchaseResult.ErrorMessage}");
}
Transfer Tokens
// Transfer tokens to another address
var transferResult = await streamingClient.Token.TransferTokensAsync(new TokenTransferRequest
{
    RecipientAddress = "0xRecipientAddress...",
    Amount = 50, // 50 tokens
    GasLimit = 150000
});

if (transferResult.Success)
{
    Console.WriteLine($"Token transfer successful!");
    Console.WriteLine($"Transaction hash: {transferResult.TransactionHash}");
}
else
{
    Console.WriteLine($"Transfer failed: {transferResult.ErrorMessage}");
}

Owner/Admin Access

For platform owners and administrators, the SDK provides advanced management capabilities that interact with the owner-level functions of your StreamingToken smart contract.

Admin Authentication
// Admin authentication requires owner wallet
var adminAuthResult = await streamingClient.Admin.AuthenticateAdminAsync(new AdminAuthRequest
{
    Address = ownerWalletAddress,
    Signature = ownerSignature,
    Message = challengeMessage
});

if (adminAuthResult.Success)
{
    // Set admin token for subsequent admin operations
    streamingClient.SetAdminToken(adminAuthResult.Token);
    Console.WriteLine("Admin authentication successful");
}
else
{
    Console.WriteLine($"Admin authentication failed: {adminAuthResult.ErrorMessage}");
}
Grant Access to User
// Grant content access to a user (admin only)
var grantResult = await streamingClient.Admin.GrantAccessAsync(new GrantAccessRequest
{
    UserAddress = "0xUserAddress...",
    ContentId = "content_123456",
    DurationInSeconds = 3600 // 1 hour
});

if (grantResult.Success)
{
    Console.WriteLine($"Access granted to user until {grantResult.ExpiryTime}");
    Console.WriteLine($"Transaction hash: {grantResult.TransactionHash}");
}
else
{
    Console.WriteLine($"Failed to grant access: {grantResult.ErrorMessage}");
}
Mint Tokens
// Mint new tokens to an address (admin only)
var mintResult = await streamingClient.Admin.MintTokensAsync(new MintTokensRequest
{
    RecipientAddress = "0xRecipientAddress...",
    Amount = 1000
});

if (mintResult.Success)
{
    Console.WriteLine($"Successfully minted 1000 tokens");
    Console.WriteLine($"Transaction hash: {mintResult.TransactionHash}");
}
else
{
    Console.WriteLine($"Failed to mint tokens: {mintResult.ErrorMessage}");
}
Get Platform Statistics
// Get platform statistics (admin only)
var stats = await streamingClient.Admin.GetPlatformStatsAsync();

Console.WriteLine($"Total users: {stats.UserCount}");
Console.WriteLine($"Total content items: {stats.ContentCount}");
Console.WriteLine($"Active streams: {stats.ActiveStreams}");
Console.WriteLine($"Total token supply: {stats.TokenSupply}");
Console.WriteLine($"Treasury balance: {stats.TreasuryBalance}");

Complete Examples

Here are some complete examples demonstrating common use cases for the Web3 Streaming .NET SDK.

Simple Streaming Application
using System;
using System.Threading.Tasks;
using Web3.Streaming.Client;
using Web3.Streaming.Client.Config;

public class StreamingExample
{
    public static async Task Main(string[] args)
    {
        // Initialize client
        var config = new Web3StreamingConfig
        {
            ApiKey = "YOUR_API_KEY",
            ApiBaseUrl = "https://api.web3streaming.com/v1",
            Network = NetworkType.Polygon
        };
        
        var client = new Web3StreamingClient(config);
        
        // Authenticate user
        await AuthenticateUser(client, "0xUserWalletAddress...");
        
        // Browse available content
        var content = await client.Content.GetContentListAsync(new ContentQueryRequest
        {
            Page = 1,
            PageSize = 10,
            Category = "education"
        });
        
        Console.WriteLine($"Found {content.TotalItems} educational content items");
        
        foreach (var item in content.Items)
        {
            Console.WriteLine($"- {item.Title} (ID: {item.ContentId})");
            
            // Check if user has access to this content
            var access = await client.Content.CheckAccessAsync(item.ContentId);
            
            if (access.HasAccess)
            {
                Console.WriteLine($"  You have access until {access.ExpiryTime}");
                
                // Get stream URL
                var streamUrl = await client.Content.GetStreamUrlAsync(item.ContentId);
                Console.WriteLine($"  Stream URL: {streamUrl}");
            }
            else
            {
                Console.WriteLine("  You don't have access to this content yet");
                
                // Purchase access if user wants to
                Console.Write("  Would you like to purchase access? (y/n): ");
                if (Console.ReadLine().ToLower() == "y")
                {
                    var startResult = await client.Streaming.StartStreamAsync(new StartStreamRequest
                    {
                        ContentId = item.ContentId
                    });
                    
                    if (startResult.Success)
                    {
                        Console.WriteLine($"  Access purchased! Valid until {startResult.ExpiryTime}");
                    }
                    else
                    {
                        Console.WriteLine($"  Purchase failed: {startResult.ErrorMessage}");
                    }
                }
            }
            
            Console.WriteLine();
        }
    }
    
    private static async Task AuthenticateUser(Web3StreamingClient client, string walletAddress)
    {
        // In a real app, you would integrate with a wallet provider
        // For this example, we'll mock the authentication
        
        Console.WriteLine($"Authenticating user {walletAddress}");
        
        // Simulate successful authentication
        client.SetAuthToken("SIMULATED_AUTH_TOKEN");
        
        Console.WriteLine("Authentication successful");
    }
}

Frequently Asked Questions

Why do I need a .NET SDK for a Web3 application?

While Web3 applications are primarily based on blockchain technologies, integrating them into traditional applications often requires seamless bridges between Web2 and Web3 technologies. Our .NET SDK provides this bridge, allowing .NET developers to easily incorporate blockchain functionality without needing to write complex Web3 integration code.

Does the SDK support cross-platform development?

Yes, our SDK is built on .NET Standard 2.0, which provides compatibility with .NET Core, .NET Framework, Xamarin, Unity, and other .NET platforms. This allows you to use the same SDK across desktop, mobile, and web applications.

How does the SDK handle blockchain transactions?

The SDK interfaces with Nethereum for blockchain interactions, providing a simplified API that abstracts away the complexity of direct blockchain calls. For advanced users, we also expose direct access to the underlying blockchain functionality.

Is the SDK suitable for backend services?

Absolutely. The SDK works well for both client and server applications. For server applications, you can use the admin APIs to manage users, content, and tokens from your backend infrastructure.
